Question 8

var vs dynamic

var → type is fixed after assignment
dynamic → can change anytime
var a = 10;
// a = "Hi"; ❌

dynamic b = 10;
b = "Hi"; // ✅

final vs const

final → value set once at runtime
const → value fixed at compile time
final time = DateTime.now();
const pi = 3.14;

Question 10

Why final works with DateTime.now():
Because it is assigned at runtime.

Why const cannot:
const requires a value known at compile time, but DateTime.now() changes every time.
